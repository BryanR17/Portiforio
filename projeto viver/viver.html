Viver Goiânia

Esse Projeto ainda não foi 100% concluido, o intuito desse projeto é ter uma rede de imoveis na cidade de Goiânia com o intuito de ajudar os moradores a encontrar seu imovel ideal, como estamos em fase de desenvolvimento e captação dos imoveis ainda, não temos tantos imoveis cadastrados, mas a ideia é que seja uma plataforma simples e intuitiva para os usuários. Para desenvolver esse projeto, utilizei HTML, CSS e JavaScript para fazer a parte visual, e utilizei o PHP para fazer a captação do formulario enviando para o email da empresa.
